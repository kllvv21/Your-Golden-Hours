# Приложение "Your Golden Hour"

Данный проект представляет собой мобильное приложение для отслеживания «Золотого часа» (идеального времени для фотосъемки). Интерфейс реализован на **Jetpack Compose**.

---

## 🏗 Инструкция по сборке APK

Для сборки проекта без использования графического интерфейса Android Studio, выполните следующие шаги:

### 1. Требования
*   Установленная Java (JDK 17 или выше).
*   Наличие Android SDK.

### 2. Сборка
Откройте терминал в корневой папке проекта и выполните команду для вашей ОС:

**Для Windows (PowerShell/CMD):**
```bash
.\gradlew clean assembleDebug
```
 **Для macOS / Linux:**
```bash
chmod +x gradlew
./gradlew clean assembleDebug
```
### 3. Результат
После успешного завершения сборки (BUILD SUCCESSFUL), файл APK будет доступен по следующему пути:
app/build/outputs/apk/debug/app-debug.apk

## Руководство пользователя

1. Назначение приложения
Приложение помогает хранить фотографии «золотого часа», смотреть их в галерее, добавлять новые снимки с данными о месте и времени, а также видеть карту с привязкой снимков к координатам и информацию о ближайшем интервале золотого часа для вашего текущего местоположения.

2. Первый запуск и условия работы
Что нужно	Зачем
Интернет	Загрузка данных о восходе/закате с внешнего API; работа картографического сервиса.
Разрешение на геолокацию	Чтобы определить ваши координаты и показать золотой час и город в шапке галереи.
Ключ Yandex MapKit (для разработчика)	Карта строится на Yandex MapKit. Ключ задаётся в сборке приложения (BuildConfig или метаданные манифеста). Без успешной инициализации на экране «Карта» будет показано сообщение об ошибке, а не карта.
Если геолокация отклонена, приложение по-прежнему откроется, но блок с временем золотого часа и городом может показать сообщение об ошибке или загрузку до тех пор, пока координаты не будут доступны.

3. Навигация по приложению
Внизу экрана — три раздела (нижнее меню):

Галерея — список карточек фотографий и блок с данными API.
Камера / добавление — форма добавления нового фото (изображение, название, дата, время, место, описание).
Карта — интерактивная карта с метками.
Экран просмотра фото и редактирования открываются из галереи по нажатию на карточку и далее по иконке редактирования; нижнее меню на этих экранах скрыто — возврат стрелкой «Назад».

4. Карта и метки с описанием
4.1. Что используется
Подключена карта Yandex MapKit (аналоги по заданию: Google Maps, OpenStreetMap и т.д.).
На карте отображаются метки (пины) для фотографий, у которых в базе сохранены координаты (широта и долгота не равны нулю одновременно). Координаты подставляются при добавлении фото, в том числе из EXIF файла, если в снимке есть GPS.
4.2. Как пользоваться
Откройте раздел «Карта».
Карта при необходимости подстраивает масштаб, чтобы были видны все фото с координатами.
Нажмите на метку — откроется карточка рядом с точкой: превью фото (можно листать, если в одной точке несколько снимков), название и время съёмки — это и есть описание на метке в пользовательском интерфейсе.
Нажатие по пустому месту карты закрывает карточку.
Пока карточка открыта, метка в этой точке скрывается, чтобы не перекрываться с «хвостом» подсказки.
Таким образом, данные п. 1 выведены в UI: сама карта, пины и всплывающая карточка с текстом и изображением.

5. Пункт задания 2 — внешний API и вывод на экран
5.1. Какой API подключён
Используется общедоступный HTTP API Sunrise Sunset (https://api.sunrise-sunset.org/): по широте, долготе и дате возвращаются время восхода и заката солнца (в формате с учётом часового пояса приложение переводит в локальное время устройства).

5.2. Где это видно пользователю
На экране «Галерея» в верхней части (шапка):

Город или населённый пункт (или подпись с координатами) — определяется по текущему местоположению через геокодирование Android после получения координат.
Интервал золотого часа в виде строки вида «ЧЧ:мм – ЧЧ:мм» — рассчитывается на основе ответа API (восход/закат и логика «час после восхода» / «час до заката» в зависимости от текущего времени суток).
При проблемах с сетью или сервером вместо интервала может отображаться короткое сообщение об ошибке (например, нет сети, таймаут).

Итог: данные внешнего API подключены и отображаются в пользовательском интерфейсе галереи, как требует задание.

6. Добавление фотографии (кратко)
Раздел камеры в нижнем меню.
Выберите изображение из галереи устройства.
При наличии GPS в файле координаты и подпись места могут заполниться автоматически; при необходимости уточните поля вручную.
Сохраните — карточка появится в галерее, а при наличии координат — метка на карте.
